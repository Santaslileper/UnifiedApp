# UnifiedApp

A premium, customizable desktop widget and wallpaper engine.

## Features
- **Modern UI**: Sleek, glassmorphic design inspired by modern aesthetics.
- **Dynamic Widgets**: Clock, Battery, Timer, Calculator, and more.
- **Custom Shortcuts**: Create desktop buttons for files, folders, and web links.
- **Live Wallpapers**: Host HTML/JS-based interactive wallpapers on your desktop.
- **Moddability**: Easily add new widgets using C# scripts (Data/Mods).
- **Volume Protection**: Limit volume per device to prevent ear damage.
- **Efficient**: Uses native Fallback renderer or Ultralight for high performance.

## Controls
- **Right-Click**: Access widget settings, theme options, and management.
- **Drag Handle**: Move widgets when unlocked.
- **Bottom-Right Grip**: Resize widgets (snaps to grid).
- **Toggle Launcher**: Use the '+' button to spawn new widgets.

## Installation
1. Extract the release ZIP.
2. Run `UnifiedApp.exe`.
3. Use the system tray icon or the '+' widget to manage your desktop.
